# Install & Run

**Please use `yarn` instead of `npm`**  

## Install package
`yarn install`

## Run This Project

`yarn run start`  

Then open `http://localhost:4200/` in browser  

(Only supports `Phantom Wallet` currently, please install browser extension first.)
